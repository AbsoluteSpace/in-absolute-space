Book Smasher Feb 2020

Features vectors are created using Bag of Words and other aspects of each sentence (such as similar sentences discovered with KMeans clustering). After a small amount of user input, a Random Forest classifies sentences before the combination is output.

  • Constructs books based on text files in C#
  • Generated features and classified examples using KMeans/Bag of Words/Random Forests

Mobile Approvals May 2019
1st Place Aucerna Company Hackathon

Created over 4 days as 3 engineers with no flutter or mobile experience. In addition to approving AFEs normally, we implemented three additional "game" modes. A Missile Command mode that approves AFEs if they reach the ground, a shakewight mode for a quick workout, and a cardstack swiping mode to meet single AFEs looking for approval in your area.

  • Developed mobile AFE app with Flutter using Android Studio in 1 week
  • Implemented card stack approval mode (think Tinder) and parsed REST APIs to personify the AFEs
