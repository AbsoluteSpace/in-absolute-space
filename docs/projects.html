Book Smasher Feb 2020

Features vectors are created using Bag of Words and other aspects of each sentence (such as similar sentences discovered with KMeans clustering). After a small amount of user input, a Random Forest classifies sentences before the combination is output.

  • Uses machine learning to combine text files into new books in C#
  • Generated features and classified examples using KMeans/Bag of Words/ Random Forests

Mobile Approvals May 2019
1st Place Aucerna Company Hackathon

Created over 4 days as 3 engineers with no flutter or mobile experience. In addition to approving AFEs normally, we implemented three additional "game" modes. One, a Missile Command mode that approves AFEs if they reach the ground, a shakewight mode for a quick workout, and a cardstack swiping mode to meet single AFEs looking for approval in your area.

  • A mobile AFE approval app built with Flutter
  • I built a cardstack approval mode (think Tinder) as well as dynamic theming
